# HR-Analytics-project-using-Power-BI
HR Analytics Dashboard created using Power BI to analyze employee data and provide insights to HR Leadership teams understand employee attrition, department-wise performance.

### Insights

Total Employees
Attrition Count & Attrition Rate
Active Employees
Average Age
Attrition by Department
Attrition by Job Role
Attrition by Gender
Attrition by Years at Company

### Tools Used

Power BI Desktop
Microsoft Excel (Dataset)
Data Cleaning & Transformation
Basic DAX
Data Visualization
